# customer_behavior_analysis
# End-to-End Data Analytics Project

## Overview

This project demonstrates a complete data analytics workflow, starting with raw data and ending with an interactive Power BI dashboard. The objective was to clean, transform, analyze, and visualize the data to generate meaningful business insights.

## Workflow

### 1. Data Cleaning and Transformation (Python - Pandas)

* Loaded the raw dataset
* Handled missing values
* Removed duplicate records
* Corrected data types
* Cleaned inconsistent data
* Performed necessary data transformations

### 2. Data Analysis (SQL)

* Imported the cleaned dataset into a SQL database
* Connected Python with the database
* Wrote SQL queries to analyze the data
* Generated business insights using filtering, aggregation, grouping, joins (if applicable), and other SQL operations

### 3. Dashboard Development (Power BI)

* Connected Power BI to the processed dataset
* Created interactive visualizations
* Designed KPIs using DAX measures
* Built a dashboard to present insights clearly and effectively
